
# Owo-style Discord Bot

## Cara Jalankan Lokal
1. Install Node.js
2. Jalankan:
```
npm install
node index.js
```

## Deploy ke Railway
1. Push ke GitHub
2. Buka railway.app → New Project → Deploy from GitHub
3. Tambahkan `.env` ke pengaturan Railway dengan isi:
   - `DISCORD_TOKEN`
   - `OWNER_ID`
